How to Play Word Bomb

Word Bomb is a fast-paced vocabulary game. A 2-3 letter fragment is shown (like "PH" or "ING"). You must quickly type a valid English word that contains that fragment. If the bomb timer runs out before you answer, you lose a life. You have 3 lives — last as many rounds as you can.

Rules

  • Type any valid word containing the shown fragment
  • Words must be at least 3 letters long
  • Each word can only be used once per game
  • Timer gets shorter as you advance — stay sharp!
