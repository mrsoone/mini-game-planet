What Is the Binary Game?

The Binary Game is a fast-paced conversion challenge that tests your understanding of binary numbers. You have 60 seconds to convert as many numbers as possible between decimal (base 10) and binary (base 2). The game starts with 4-bit numbers (0–15) to warm you up, then progresses to 8-bit numbers (0–255) for a tougher challenge. Each correct conversion earns points, and consecutive correct answers build a streak multiplier that boosts your score. A visual bit display shows the place values (128, 64, 32, 16, 8, 4, 2, 1) so you can see exactly how each binary digit contributes to the total.

How It Works

Choose your mode before starting: Decimal to Binary or Binary to Decimal. In Decimal-to-Binary mode, you'll see a decimal number like 13 and must type its binary representation—1101. In Binary-to-Decimal mode, you'll see a binary number like 1101 and must type the decimal equivalent—13. Type your answer and press Enter to submit. The game uses crypto.getRandomValues() to generate every number, ensuring fair, unpredictable challenges. No patterns, no repetition bias—just pure random conversion practice.

4-Bit and 8-Bit Modes

For the first 30 seconds, the game uses 4-bit numbers. In binary, 4 bits can represent values from 0 to 15. The bit positions are 8, 4, 2, and 1. For example, the decimal number 13 equals 8 + 4 + 1, so in binary it's 1101. After 30 seconds, the game switches to 8-bit mode. Eight bits can represent 0 to 255, with place values 128, 64, 32, 16, 8, 4, 2, and 1. The decimal number 255 is 11111111 in binary—every bit is 1. The visual display shows these place values so you can quickly see which bits are set.

Scoring and Streaks

Each correct answer earns base points. Your streak multiplier starts at 1× and increases by 1 for every consecutive correct answer. A 5-question streak means 5× points per correct answer. Miss one and the streak drops back to 1. This rewards both speed and consistency. Your personal best score is saved in your browser's localStorage, so you can track improvement over time. Session statistics show how many conversions you solved correctly in the current browsing session.

Why Learn Binary?

Binary is the language of computers. Every piece of data—text, images, sound—is ultimately stored as sequences of 0s and 1s. Understanding binary conversion helps you grasp how computers work, how memory is addressed, and how colors and numbers are encoded. Programmers, computer science students, and anyone curious about technology benefit from fluency in binary. The Binary Game turns this essential skill into a timed challenge that sharpens your mental conversion speed.

Conversion Tips

  • Memorize powers of 2. 1, 2, 4, 8, 16, 32, 64, 128—these are your building blocks. Decimal 13 = 8 + 4 + 1, so binary 1101.
  • Work from largest to smallest. For decimal to binary, subtract the largest power of 2 that fits, mark that bit as 1, repeat.
  • Binary to decimal is addition. For 1101, add 8 + 4 + 0 + 1 = 13. Only add the place values where the bit is 1.
  • Leading zeros don't matter. 1101 and 00001101 both equal 13. The game accepts either format.
